# Prompt
[무엇을] 코딩 경험이 없는 초보자용으로 "키와 몸무게 골격근량을 3대 웨이트 운동의 적정한 무계계산을 알려주는 웹을 만들꺼야", 제목은 "본인의 3대 웨이트 운동의 적정 중량"
[조건]
-필요한 개인의 기본 신체 정보는 "체중, 신장, 성별, 나이"야- 
-개인의 운동 경력을 입력할껀데 "3,6,9,12개월, 그 이상 단위로 5가지의 경우를 입력받음 또한 동작 숙련도는 초급 중급 고급으로 설정"-
-현재 수행 가능 기록은 "몇 키로로 몇 세트 가능한지를 입력을 통해 1RM을 측정함","예를 들어 "현재 벤치프레스 최대@KG으로 @회 @세트 가능"-
-목표 설정 "벌크업, 다이어트, 기초 체력 향상 총 3가지로 설정"-
-부상 이력 작성,"있음과 없음" 두가지 경우로 설정-
[스타일] 약간 운동 동기부여가 되는 느낌의 스타일로 해주고 색깔은 헬스장에서 많이 사용하는 회색 및 검은색 바탕이 좋을듯해

[마무리] 실행방법(어떤 파일을 열면 되는지)와 "벤치프레스,데드리프트,스쿼트" 적정 중량으로 몇회,몇세트를 알려주고 수정포인트 3가지를 알려줘

[내용]
-기본 신체 정보 입력: 체중, 신장, 숙련도, 골격근량을 외부로 부터 입력받음
-각 3대 운동의 몇 KG으로 몇 개를 몇 세트 가능한지를 외부로 부터 받은 데이터로 계산해서 알려줘
-목표 설정을 통해 최대 몇 KG로 몇개를 몇 세트 가능한지 또한 알려주면 좋을껏 같아

<1월 29일 만든 프로그램을 통해 추가적으로 덧붙이고 싶은 내용>
-선택사향으로 인바디를 통해 본인이 측정한 현재 본인의 골격근량(KG) 과 체지방률(%) 도 적어주면 좋을듯 하다. 
-CURRENT RECORD에서 한 세트 내에서 본인이 할수 있는 최대 중량(KG)으로 몇 회 가능한지로 적으라고 알려주는게 좋을듯 하다.
-각 목표별 (벌크업,기초체력향상,다이어트)로 본인이 하루에 섭취해야할 적정 순탄수화물(g),단백질(g),지방(g)도 알려주면 좋을듯 하다
-본인이 각 목표별(벌크업, 기초체력향상, 다이어트)를 실행할시 본인이 피해야할 식습관이나 자제해야할 음식들도 알려주면 좋을듯 하다.
-본인의 데이터를 기반하여 본인의 목표별 적정 웨이트 운동시간 및 유산소 운동 시간 또한 알려주면 좋을듯 하다.
